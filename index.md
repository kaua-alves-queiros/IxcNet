# Bem-vindo à documentação do **IxcNet**

Esta é a biblioteca de integração com a API do IXCSoft. Aqui você encontrará referências técnicas para as classes, métodos e modelos utilizados no projeto.

## Começo Rápido:

1. Explore a [API](api/IxcNet.Services.yml) para entender como realizar consultas.
2. Veja os [ViewModels](api/IxcNet.ViewModels.Sistema.Cadastros.yml) para conhecer as estruturas de dados retornadas pela API.
3. Se precisar de ajuda, consulte a documentação técnica oficial do IXCSoft ou entre em contato com a equipe de desenvolvimento.
